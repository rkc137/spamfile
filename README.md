# spamfile
a little program to fill ur drive with copys of single file for fun :D (i love to gift my friends drives with our local jokes on it, like a million copys of one gif)
#### usage
```
spamfile [options]
    -sf: path to source file
    -tf: path to target folder
    -s: size in bytes (if missing gonna use all avalible space)
    -sm: same as -s but in MB
    -h: list of options
```
